Use build.py o buildUncompressed.py to generate a single js file for all ORS js-files.

HowTo:
1) Modify ors.cfg to contain all necessary ORS js-files in the correct order.
2) Make sure license.txt contains the correct license, authors and version number.
3) run build.py or buildUncompressed.py from the command line (no additional parameters necessary)
4) copy or link the generated ORS.js file to the application.
